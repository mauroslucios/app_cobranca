# App_cobrança
### Gerenciamento de cobrança endpoints
- POST-> cadastrar um título com seu status inicial de PENDENTE
- UPDATE-> atualizar um título de PENDENTE  para RECEBIDO
- GET-> visualizar todos os títuos
- DELETE-> apaga um título
